What is OpenVAS?

OpenVAS (Open Vulnerability Assessment System) is a full-featured open-source vulnerability scanner used to find security holes in systems and networks. It is widely used by security professionals for vulnerability assessment and compliance auditing.

Key Features

  • Powerful vulnerability scanning engine
  • Supports thousands of test plugins
  • Web-based management with Greenbone Security Assistant (GSA)
  • Scheduled scans and detailed reports
  • Supports CVE scanning and compliance checks

Installation

On Kali Linux:

sudo apt update
sudo apt install openvas
sudo gvm-setup
sudo gvm-check-setup

Starting OpenVAS

sudo gvm-start

Once running, access the web interface (GSA) at:

https://localhost:9392

Step-by-Step: Scanning with OpenVAS

  1. Start OpenVAS: sudo gvm-start
  2. Open your browser and go to: https://localhost:9392
  3. Log in using the credentials provided during setup
  4. Go to Scans ‚Üí Tasks and click "New Task"
  5. Enter a task name and the target IP/domain
  6. Click "Save" and then "Start Scan"
  7. After scan completes, view the detailed report

Use Cases

  • Enterprise vulnerability assessment
  • Network security audits
  • Penetration testing preparation
  • Security compliance verification (PCI-DSS, etc.)

Important Tips

  • Scan with permission to avoid legal issues
  • Keep the vulnerability feed up to date
  • Combine OpenVAS results with manual testing
