What is Wireshark?

Wireshark is a powerful, open-source network protocol analyzer. It allows cybersecurity professionals to capture and inspect data packets in real-time, helping to troubleshoot networks, analyze traffic, and detect suspicious activity.

Why Use Wireshark?

  • To analyze live network traffic
  • To detect malicious activity
  • To troubleshoot connectivity issues
  • To study protocols and learn networking
  • To capture passwords (in insecure environments)

Installing Wireshark (Linux or Windows)

For Linux (Debian-based):

  1. Update packages:
    sudo apt update
  2. Install Wireshark:
    sudo apt install wireshark
  3. Allow non-root users:
    sudo dpkg-reconfigure wireshark-common
    sudo usermod -aG wireshark $USER
  4. Restart your system.

For Windows:

  1. Download from: Wireshark Official Website
  2. Install and choose to include WinPcap or Npcap during setup
  3. Launch and start capturing!

Basic Usage Steps

  1. Open Wireshark
  2. Select the network interface (Wi-Fi, Ethernet, etc.)
  3. Click "Start" to begin capturing packets
  4. Use filters like:
    http ‚Äî show only HTTP traffic
    ip.addr == 192.168.1.1 ‚Äî filter by IP address
  5. Click on any packet to view details
  6. Save captures as `.pcap` files for later analysis

Useful Filters

  • tcp ‚Äî Show TCP packets
  • udp ‚Äî Show UDP packets
  • dns ‚Äî Show DNS traffic
  • ftp ‚Äî Show FTP traffic
  • http.request ‚Äî Show only HTTP requests

Advanced Tips

  • Capture passwords from insecure protocols like FTP or HTTP
  • Follow a TCP stream to see full conversation:
    Right-click ‚Üí Follow ‚Üí TCP Stream
  • Export packet data or objects (like files or images)
  • Use color rules to highlight suspicious traffic
  • Set capture filters before capturing for better performance

Warning

  • Do not capture traffic on networks you don't own or have permission to monitor
  • Use in ethical hacking labs or with explicit authorization
