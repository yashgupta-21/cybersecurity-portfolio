What is Ettercap?

Ettercap is a powerful open-source network security tool used for man-in-the-middle (MITM) attacks on LAN. It supports active and passive dissection of many protocols and includes features for network sniffing, host discovery, and ARP poisoning.

Key Features

  • ARP poisoning and spoofing
  • DNS spoofing
  • Packet filtering
  • Real-time connection interception
  • Plugin support and GUI mode

Installation

On Kali Linux / Debian:

sudo apt update
sudo apt install ettercap-graphical

Verify Installation:

ettercap -v

Launching Ettercap

Ettercap can run in both CLI and GUI mode:

sudo ettercap -G

This opens the graphical interface.

Step-by-Step: MITM Attack Using Ettercap

  1. Open terminal and run:
    sudo ettercap -G
  2. Select Sniff ‚Üí Unified Sniffing ‚Üí Choose your network interface (e.g., eth0 or wlan0)
  3. Go to Hosts ‚Üí Scan for Hosts
  4. Go to Hosts ‚Üí Hosts List, then add Target 1 and Target 2 (usually router and victim)
  5. Click Mitm ‚Üí ARP Poisoning
  6. Select "Sniff remote connections"
  7. Go to Start ‚Üí Start Sniffing

Example CLI MITM Command

sudo ettercap -T -M arp:remote /192.168.1.1/ /192.168.1.100/

This command performs MITM between the router (gateway) and the victim.
