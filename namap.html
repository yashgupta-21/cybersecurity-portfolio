What is Nmap?

Nmap (Network Mapper) is a free and open-source tool for network discovery and security auditing. It‚Äôs used to discover hosts, open ports, services, OS details, and vulnerabilities in a network.

Why Use Nmap?

  • To scan networks and IP ranges
  • To identify open ports and services
  • To detect the operating system and version
  • To find vulnerabilities in networks
  • To audit firewall rules and configurations

How to Install Nmap in Termux

  1. Update packages:
    pkg update && pkg upgrade
  2. Install Nmap:
    pkg install nmap
  3. Check version:
    nmap --version

Basic Nmap Commands

  • Scan a single IP:
    nmap 192.168.1.1
  • Scan a range of IPs:
    nmap 192.168.1.1-100
  • Scan an entire subnet:
    nmap 192.168.1.0/24
  • Scan multiple ports:
    nmap -p 22,80,443 192.168.1.1
  • Scan all 65535 ports:
    nmap -p- 192.168.1.1
  • Service version detection:
    nmap -sV 192.168.1.1
  • Operating system detection:
    nmap -O 192.168.1.1
  • Aggressive scan:
    nmap -A 192.168.1.1

Advanced Nmap Scanning Examples

  • Scan with output saved to file:
    nmap -oN scan.txt 192.168.1.1
  • Detect firewall:
    nmap -sA 192.168.1.1
  • UDP Scan:
    nmap -sU 192.168.1.1
  • Scan with NSE scripts:
    nmap --script vuln 192.168.1.1

Top Uses of Nmap

  • Find live hosts on a network
  • Discover open ports and running services
  • Identify software versions and vulnerabilities
  • Monitor network security
  • Test firewalls and security policies

Important Tips

  • Always scan ethically ‚Äî never without permission
  • Try Nmap in labs, CTFs, and test environments
  • Combine Nmap with other tools for deeper results
