What is OWASP ZAP?

OWASP ZAP (Zed Attack Proxy) is one of the most popular open-source web application security scanners. It helps find security vulnerabilities in web apps during development and testing.

Key Features

  • Automated web vulnerability scanner
  • Intercepting proxy for manual testing
  • Spider and AJAX spider
  • Active and passive scanning
  • Add-on support for extra features

Installation

On Kali Linux / Debian:

sudo apt update
sudo apt install zaproxy

Using Snap (Any Linux):

sudo snap install zaproxy

On Windows/macOS:

Download from: https://www.zaproxy.org/download/

Launching OWASP ZAP

zaproxy

This will open the graphical interface of ZAP.

Manual Testing with Proxy

  • Set browser proxy to 127.0.0.1:8080
  • Browse your target website
  • ZAP will capture all HTTP requests and responses
  • Use the request editor to manipulate parameters

Common Vulnerabilities Detected

  • XSS (Cross-site scripting)
  • SQL Injection
  • Insecure cookies
  • Information disclosure
  • Missing security headers

Important Tips

  • Run only on apps you own or have permission to test
  • Use in dev or staging environments, not live production
  • Combine with manual testing for best results
