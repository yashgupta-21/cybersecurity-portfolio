What is Snort?

Snort is an open-source Intrusion Detection System (IDS) and Intrusion Prevention System (IPS). It analyzes network traffic in real-time and can detect suspicious or malicious activity using defined rules and signatures.

Key Features

  • Real-time traffic analysis
  • Packet logging
  • Protocol analysis
  • Detect port scans, buffer overflow attacks, malware, etc.
  • Highly customizable rule sets

Installation

On Kali Linux:

sudo apt update
sudo apt install snort

Verify Installation:

snort -V

Snort Modes

  • Sniffer Mode ‚Äì Reads packets from the network and displays them
  • Packet Logger Mode ‚Äì Logs packets to a file
  • NIDS Mode ‚Äì Detects intrusions using defined rules

Step-by-Step Usage

1. Sniffer Mode:

sudo snort -v

This shows all incoming packets in real-time.

2. Packet Logger Mode:

sudo snort -dev -l /path/to/log

Logs packets to the specified directory.

3. IDS Mode:

sudo snort -A console -q -c /etc/snort/snort.conf -i eth0

-A: Alert mode, -i: interface

Writing Basic Snort Rules

Example Rule:

alert tcp any any -> any 80 (msg:"HTTP Access"; sid:1000001;)

This alerts on any TCP traffic to port 80 (HTTP).

Log Analysis

Logs are stored in the path you specify. Use tools like Wireshark or manually inspect alert.ids files for suspicious entries.

Important Files

  • /etc/snort/snort.conf ‚Äì Main configuration file
  • /etc/snort/rules/ ‚Äì Rule files directory

Snort Use Cases

  • Home or small business network monitoring
  • Detect brute-force attacks, DoS, SQL injection
  • Integration into SIEM tools
