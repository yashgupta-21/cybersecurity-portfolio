What is John the Ripper?

John the Ripper is a fast and powerful password cracking tool. It is used to perform dictionary attacks and brute-force attacks on hashed passwords from shadow files, zip files, documents, and more.

Key Features

  • Supports many hash formats (MD5, SHA, etc.)
  • Cracks Linux, Windows, and encrypted archive passwords
  • Highly customizable and fast
  • Includes built-in wordlist and rules

Installation

On Termux:

pkg install unstable-repo
pkg install john

On Kali Linux:

sudo apt install john

Verify Installation:

john --help

Step-by-Step Usage

  1. Collect the hashed password file (e.g., from shadow file)
  2. Use unshadow to merge /etc/passwd and /etc/shadow:
    3. unshadow passwd.txt shadow.txt > merged.txt
  3. Start cracking with:
    4. john merged.txt
  4. To show cracked passwords:
    5. john --show merged.txt

Example: Cracking ZIP File Password

zip2john secret.zip > hash.txt
john hash.txt

Supported File Types

  • /etc/shadow (Linux password hash)
  • ZIP, RAR, 7z archive passwords
  • PDF documents
  • Word/Excel documents
  • Raw MD5, SHA1, bcrypt, etc.

Tips

  • Use --wordlist= to specify custom dictionary
  • Use rules to mutate passwords for better results
  • Combine with hashcat for GPU-based cracking
